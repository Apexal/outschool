What are webpages?

A webpage is a single page you visit in your web browser. It can have information, links, text, videos, sound, moving parts, etc. Usually these are on the web.

What is the web?

The world wide web is the huge, huge, huge collection of webpages that are publicly available on the internet. Links usually connect them.

HTML Elements

HTML pages are made up of elements which usually have text inside of them. Different elements have different meanings. Some are paragraphs, some are headers, some are lists, some are images, etc.

HTML Elements

Let's look at some basic elements.

HTML Tags

When you first start to learn HTML, you need to learn what tags there are!

<p>paragraph</p>

The <p> tag goes around text that should be its own paragraph.

<strong>bold</strong>

The <strong> tag goes around text that should be bolded. You can also use the older <b> tag for this.

<em>emphasis</em>

The <em> tag goes around text that should be italicized. You can also use the older <i> tag for this.

<h1>heading</h1>

The <h1> tag goes around text that should be a big heading! There are 6 of these. <h1> is the largest and <h6> is the smallest.

<a>clickable link</a>

The <a> tag goes around text that should be a clickable link! You have to say what to link to and what text to show.

Tags Inside Tags Inside Tags...

The whole point of HTML is that tags usually go inside of each other!
Some tags cannot go inside others, but that's for another time...

Putting It All Together

You can make a full webpage with just these tags!

Full Example


                        <!-- This a comment. It is ignored by the webpage. It's just for us to leave messages in the code. -->

                        <!-- This tells us what language we are using... HTML of course! -->
                        <!DOCTYPE html>

                        <!-- Everything MUST be inside the <html> tag. -->
                        <html>

                            <!-- The <head> has information about the website that doesn't show when you look at it. -->
                            <head>
                                <title>My Website</title>
                            </head>

                            <!-- The body is where the visible stuff goes! -->
                            <body>
                                <h1>My Cool Website</h1>
                                <p>Well hello there and welcome to my cool website.</p>
                            </body>

                        </html>
